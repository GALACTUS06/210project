# 3-5 Minute Presentation Script

## Slide 1 - Title

Hello everyone. Our project is a simple, interactive misinformation-disinformation flowchart website built as a linear storyline.

The goal is to help users clearly follow how information can move from a verified fact to public consequences, and then to correction.

## Slide 2 - Research Focus

Our research asks a practical question: why does false information spread quickly, and why does it still influence people even after correction?

Using interdisciplinary sources, we examined psychological, social, and platform mechanisms behind misinformation and disinformation. The final site translates that research into a step-by-step learning experience.

## Slide 3 - Conceptual Flow

The main path is:

Fact -> Mishearing -> Propagation -> Amplification -> Public Reaction -> Correction

Each node contains:

- a concise 2-3 sentence definition
- a real-world example
- supporting citations from our bibliography
- an illustration placeholder

This makes the flow easy to understand for first-time users.

## Slide 4 - Evidence Base

Each stage is supported by the annotated bibliography.

For example:

- Vosoughi et al. explains rapid spread of false news
- Lewandowsky et al. explains persistence after debunking
- Ferreira Caceres et al. and Pew show social and health consequences
- Wardle and Derakhshan provide the information disorder framework

So the flowchart is not just visual. It is evidence-backed and citation-linked.

## Slide 5 - Website Interaction

The interface is intentionally simple and linear:

- users click Next and Previous to follow the storyline
- users can jump to any node from the sidebar list
- the current stage is highlighted in both sidebar and path track
- transitions are smooth so movement through stages feels guided

All content is loaded from a single JSON file, so updates are easy without rewriting UI logic.

## Slide 6 - Accessibility and Usability

The project includes keyboard and screen-reader considerations:

- clear aria labels on controls
- keyboard navigation with Left/Right arrows
- live status text for current step announcements
- images include alt text
- layout is responsive on mobile and desktop

We also added a theme toggle to improve readability under different viewing preferences.

## Slide 7 - Why This Matters

Misinformation is not only a content problem. It is a process problem.

By showing each step clearly, this project helps users identify intervention points and discuss digital literacy, trust, and media behavior.

## Slide 8 - Future Extensions

Next steps can include:

- adding optional branch nodes for manipulation and fabrication
- adding quiz prompts after each node
- adding multilingual node text
- adding downloadable classroom worksheets

Because the architecture is JSON-driven, each extension can be added incrementally.

## Closing

In summary, this project turns our research into a clean, interactive, and accessible learning tool. The linear structure makes the misinformation process easier to follow while keeping every stage grounded in credible sources.

Thank you.
